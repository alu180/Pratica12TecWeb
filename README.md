# 📚 Minha Biblioteca

fiz a aplicação web feita como exercício da disciplina de Tecnologia Web. Permite criar uma conta, fazer login e gerenciar uma lista pessoal de livros favoritos.


 Como usar o sistema

 1 Página de Login (`index.html`)
A aplicação começa na tela de login. Se você ainda não tem uma conta, clique em **"Cadastre-se aqui"** para criar uma.

 2 Página de Cadastro (`cadastro.html`)
Preencha um nome de usuário (mínimo 3 caracteres) e uma senha (mínimo 4 caracteres). Após cadastrar, você será redirecionado automaticamente para o login.

 3 Página de Listagem (`listagem.html`)
Após fazer login, você verá sua lista com 9 livros favoritos pré-cadastrados. Nessa página você pode:

- Editar um livro — clique no botão "Editar" ao lado do livro desejado, altere as informações no formulário e clique em "Salvar"
- Remover um livro — clique no botão "Remover" e confirme a exclusão
- Adicionar um livro — preencha o título e o autor no formulário no final da página e clique em "Salvar"
- Sair — clique em "Sair" no canto superior direito para voltar à tela de login
